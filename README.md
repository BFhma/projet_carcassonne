# Projet Carcassonne

Ce projet est une version simplifiée du jeu de société Carcassonne, adaptée pour un projet de groupe.  
Il permet de jouer en multijoueur.

## Objectif du projet

- Permettre le jeu multijoueur jusqu’à 5 joueurs   
- Poser des tuiles pour construire des routes, villes et abbayes  
- Placer des meeples sur les éléments du jeu pour marquer des points
